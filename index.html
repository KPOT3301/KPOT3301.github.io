🚫 Правила использования

  • Не использовать с российскими сервисами (Яндекс, ВК, Mail.ru) при включенном соединении — сломаете конфиг.
  • Не передавать конфиги всем подряд, только через бота.
  • Не продавать и не распространять ключи.
  • Не использовать торренты.
  • Не скачивать файлы свыше 100+ ГБ.
  • Не совершать противоправные действия на территории РФ.
